# Port Numbers - Security+ Notes

## Common Ports
- 21 - FTP
- 22 - SSH
- 23 - Telnet
- 25 - SMTP
- 53 - DNS
- 80 - HTTP
- 110 - POP3
- 143 - IMAP
- 161 - SNMP
- 443 - HTTPS

## Tricky ones to remember
- IMAP = 143 (retrieves email, keeps it on server)
- SNMP = 161 (network device monitoring, NOT email-related)
